### 상세 문항 정보

#### q1010 - 반응 확인
- **질문**: 쓰러진 환자를 발견한 당신이 가장 먼저 해야 할 행동은?
- **선택지**: 1.✅환자의 반응을 확인한다 / 2.선임간호사에게 도움 요청 / 3.바로 눕혀 기도 확보 / 4.진정제 PRN 확인
- **정답 피드백**: 정확합니다! 환자의 반응을 확인하는 것이 초기 대응의 첫 단계입니다.
- **질문 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365693/cpr-videos/q1010_question.mp4`

#### q1020 - 반응 확인 방법
- **질문**: 환자의 반응을 확인하는 방법은 무엇인가요?
- **선택지**: 1.큰소리로 말을 걸어본다 / 2.얼굴을 들여다보며 호흡 소리를 듣는다 / 3.✅어깨를 두드리며 '괜찮으세요?' 묻는다 / 4.어깨를 세게 흔들며 이름을 부른다
- **정답 피드백**: 어깨를 두드리며 큰소리로 묻는 것이 표준화된 반응 확인 방법입니다.

#### q1030 - 도움 요청
- **질문**: 환자의 반응이 없습니다. 어떻게 해야 하나요?
- **선택지**: 1.✅도움을 요청한다 / 2.혈당을 측정한다 / 3.환자를 침대로 옮긴다 / 4.가슴압박을 시작한다
- **정답 피드백**: 정확합니다! 환자의 반응이 없으면 즉시 주변에 도움을 요청해야 합니다.
- **질문 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365699/cpr-videos/q1030_question.mp4`

#### q1040 - 도움 요청 방법
- **질문**: 도움 요청을 어떻게 해야 할까요?
- **선택지**: 1.간호사실에 가서 보고 / 2.핸드폰으로 선임에게 전화 / 3.보호자에게 간호사 부르라고 함 / 4.✅호출벨을 눌러 주변에 알린다
- **정답 피드백**: 도움요청을 위해 호출벨을 누르는 것이 가장 빠른 방법입니다.

#### q1050 - 맥박 및 호흡 확인
- **질문**: 도움 요청 후 무엇을 해야 하나요?
- **선택지**: 1.✅맥박과 호흡을 10초 이내로 확인 / 2.모니터 부착 / 3.정맥 주사라인 확보 / 4.산소 적용
- **정답 피드백**: 정확합니다! 도움 요청 후에는 환자의 맥박과 호흡을 10초 이내로 확인해야 합니다.
- **질문 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365703/cpr-videos/q1050_question.mp4`

#### q1060 - 맥박/호흡 확인 방법
- **질문**: 맥박과 호흡은 어떻게 확인하나요?
- **선택지**: 1.✅경동맥에 두 손가락 + 가슴 움직임과 호흡음 관찰 / 2.손목 요골맥 + 코에 손 대기 / 3.복부에 귀 대기 / 4.심전도 모니터 부착
- **정답 피드백**: 정답입니다! 경동맥 촉지와 흉곽 움직임, 호흡음 확인이 표준 절차입니다.
- **정답 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365708/cpr-videos/q1060_answer.mp4`

#### q1070 - 호흡 양상 판단
- **질문**: 이 환자의 호흡 상태는 어떤 양상을 보이나요?
- **선택지**: 1.정상 호흡 / 2.✅심정지호흡(agonal breathing) / 3.수면성 호흡 / 4.과호흡
- **정답 피드백**: 정확합니다! 영상에서 보이는 간헐적 헐떡임은 심정지 호흡(agonal breathing)입니다.
- **질문 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365712/cpr-videos/q1070_question.mp4`

#### q1080 - 초기 조치 결정

- **선택지**: 1.E-cart 가져오기 / 2.주치의 연락 후 대기 / 3.모니터 부착 / 4.✅가슴압박을 시작한다
- **정답 피드백**: 정답입니다! 심정지 호흡이 관찰되면 즉시 가슴압박을 시작해야 합니다.

#### q1090 - 가슴압박 위치
- **질문**: 가슴압박 시 올바른 압박 위치는 어디인가요?
- **선택지**: 1.✅흉골 중앙 하단부, 양쪽 유두 사이 / 2.왼쪽 가슴 위 손끝으로 / 3.배 위(명치) / 4.흉골 3~4회만
- **정답 피드백**: 올바릅니다! 흉골 중앙의 하단부, 양쪽 유두 사이를 압박해야 합니다.
- **질문 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365726/cpr-videos/q1090_question.mp4`

#### q1100 - 가슴압박 깊이
- **질문**: 가슴압박의 올바른 방법은 무엇인가요?
- **선택지**: 1.✅5cm 깊이 + 완전히 이완 / 2.5cm 깊이 + 빠르게 지속 / 3.10cm 이상 / 4.빠르게만
- **정답 피드백**: 정답입니다! 5cm 깊이와 완전 이완(recoil)이 중요합니다.

#### q1110 - 가슴압박 속도
- **질문**: 가슴압박의 올바른 속도는 얼마인가요?
- **선택지**: 1.✅100~120회/분 / 2.50회/분 / 3.150회/분 / 4.가능한 빠르게
- **정답 피드백**: 잘했습니다! 100~120회/분은 최적 속도입니다.

#### q1120 - 가슴압박 자세
- **질문**: 가슴압박 시 신체 자세로 옳은 것은?
- **선택지**: 1.팔꿈치 굽히고 손목 힘 / 2.손가락 끝으로 / 3.✅팔을 곧게 펴고 어깨가 흉부 위에 수직 / 4.허리 굽혀 상체만
- **정답 피드백**: 정답입니다! 팔을 곧게 펴고 어깨를 환자 흉부 위에 수직으로 위치시키면 효율적으로 압박할 수 있습니다.
- **정답 영상**: `https://res.cloudinary.com/dn3cicucf/video/upload/v1767365732/cpr-videos/q1120_answer.mp4`

## 주요 학습 포인트

### CPR 기본 순서
1. **반응 확인**: 어깨를 두드리며 "괜찮으세요?" 질문
2. **도움 요청**: 호출벨을 눌러 주변에 알림
3. **맥박/호흡 확인**: 경동맥 촉지 + 흉곽 움직임 관찰 (10초 이내)
4. **심정지 판단**: 무호흡 또는 심정지호흡(agonal breathing) 확인
5. **가슴압박 시작**: 즉시 고품질 CPR 시작

### 가슴압박 핵심
- **위치**: 흉골 중앙 하단부 (양쪽 유두 사이)
- **깊이**: 5cm
- **속도**: 100~120회/분
- **자세**: 팔을 곧게 펴고 어깨가 환자 흉부 위에 수직
- **완전 이완**: 압박 후 흉곽이 완전히 이완되도록 함 (recoil)

### 제세동기 준비 순서
1. 전원 켜기 (A)
2. 전극 부착 (C)
3. 리듬 확인 (B)
4. Shockable 리듬 시 충전 (D)

### 에피네프린 투여
- **용량**: 1mg IV bolus
- **간격**: 3~5분마다 반복
- **투여 후**: 생리식염수로 flush
- **Shockable 리듬**: 제세동 후 가슴압박 중 투여
- **Non-shockable 리듬**: 가능한 빨리 투여

### 팀 CPR 역할 분담
- **간호사 1**: 반응/맥박/호흡 확인, 가슴압박
- **간호사 2**: CPR팀 호출, 산소공급, 기도관리
- **간호사 3**: E-cart 준비, 제세동기, 기관삽관 물품 준비
