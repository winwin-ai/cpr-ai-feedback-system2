# CPR 교육 플랫폼 (전남대학교 간호대학)

## 프로젝트 소개

전남대학교 간호대학 조인영 교수님이 주도하는 연구 프로젝트로, 실제 응급 상황 시나리오를 제공하고 학습자의 판단에 대해 실시간 피드백을 제공하는 인터랙티브 CPR 교육 플랫폼입니다.

### 개발 정보

- **개발 기관**: 전남대학교 간호대학
- **연구 책임자**: 조인영 교수
- **저작권**: © 2025 전남대학교 간호대학. All rights reserved.

## 주요 기능

- **실감나는 시나리오**: 고화질 시나리오 영상을 통해 몰입감 있는 학습 환경을 제공합니다.
- **단계별 학습 (Session)**:
  - **Session 1**: 심정지 인지 및 초기 대응 (현장 안전, 의식 확인, 신고)
  - **Session 2**: 고품질 심폐소생술(CPR) 및 자동심장충격기(AED) 사용법
- **실시간 피드백**: 학습자의 선택에 따라 즉각적인 정답/오답 피드백과 상세한 해설을 제공합니다.
- **성과 분석**: 세션 종료 후 점수와 합격 여부를 판정하고, 부족한 부분을 재학습하도록 유도합니다.

## 교육 커리큘럼 및 시나리오 상세

이 플랫폼에 탑재된 모든 교육 콘텐츠는 다음과 같습니다.

### 교육 시나리오 및 문항 (Q1-Q22)

#### Q1. 초기 대응 확인

- **질문**: 쓰러진 환자를 발견한 당신이 가장 먼저 해야 할 행동은?
- **정답**: **a) 환자의 반응을 확인한다.**
- **해설**: 쓰러진 환자 발견 시 가장 먼저 수행해야 할 단계는 의식 확인입니다.

#### Q2. 반응 확인 방법

- **질문**: 환자의 반응을 확인하는 방법은 무엇인가요?
- **정답**: **c) 환자의 어깨를 두드리며 큰소리로 "괜찮으세요"라고 묻는다.**
- **해설**: 반응 확인은 양쪽 어깨를 가볍게 두드리며 큰 목소리로 물어보는 것이 표준 절차입니다.

#### Q3. 무반응 시 대처

- **질문**: 환자의 반응이 없습니다. 어떻게 해야 하나요?
- **정답**: **a) 도움을 요청한다.**
- **해설**: 환자가 반응이 없다면 즉시 주변에 도움을 요청하여 의료 응급팀(Code Blue 등)을 활성화해야 합니다.

#### Q4. 도움 요청

- **질문**: 도움요청을 어떻게 해야 할까요?
- **정답**: **d) 호출벨을 눌러 주변에 알린다.**
- **해설**: 병실 내 호출벨을 사용하여 즉시 동료 의료진에게 상황을 전파해야 합니다.

#### Q5. 도움 요청 후 조치

- **질문**: 도움요청 후 무엇을 해야 하나요?
- **정답**: **a) 환자의 맥박과 호흡을 10초 이내로 확인한다.**
- **해설**: 심정지 여부를 판단하기 위해 10초 이내에 맥박과 호흡 유무를 동시에 확인해야 합니다.

#### Q6. 맥박과 호흡 확인

- **질문**: 맥박과 호흡은 어떻게 확인하나요?
- **정답**: **a) 경동맥에 두 손가락을 대어 맥박을 느끼며 동시에 가슴의 움직임과 호흡음을 관찰한다.**
- **해설**: 경동맥 촉지와 흉부 상승 관찰을 동시에 수행하여 5~10초 이내에 완료해야 합니다.

#### Q7. 호흡 양상 판단

- **질문**: 이 환자의 호흡 상태는 어떤 양상을 보이나요?
- **정답**: **b) 심정지호흡을 보임 / 또는 간헐적 헐떡임(agonal breathing) 양상을 보임**
- **해설**: 심정지 직후 나타나는 헐떡임(Agonal breathing)은 정상 호흡이 아니며, 즉시 심정지로 간주해야 합니다.

#### Q8. 호흡 고려 조치

- **질문**: 환자의 호흡을 고려했을 때, 당신은 어떤 조치를 취해야 할까요?
- **정답**: **d) 가슴압박을 시작한다.**
- **해설**: 비정상 호흡(심정지 호흡)과 맥박 부재 시 즉시 흉부 압박을 시작하는 것이 생존율을 높입니다.

#### Q9. 가슴 압박 위치

- **질문**: 가슴압박 시 올바른 압박 위치는 어디인가요?
- **정답**: **a) 흉골 중앙의 하단부(양쪽 nipple 사이)에 손바닥을 올리고 압박한다.**
- **해설**: 흉골의 하부 1/2 지점(양쪽 젖꼭지 연결선의 중앙)이 정확한 압박 위치입니다.

#### Q10. 가슴 압박 방법

- **질문**: 가슴압박의 올바른 방법은 무엇인가요?
- **정답**: **a) 5cm 깊이로 압박하고 압박 후에는 흉곽이 완전히 이완되도록 한다.**
- **해설**: 성인 기준 약 5cm 깊이로 강하게 압박하고, 각 압박 후 가슴이 완전히 올라오도록 이완해야 합니다.

#### Q11. 가슴 압박 속도

- **질문**: 가슴압박의 올바른 속도는 얼마인가요?
- **정답**: **a) 100~120회/분으로 압박한다.**
- **해설**: 가슴압박 속도는 분당 100~120회가 권장됩니다.

#### Q12. 가슴 압박 자세

- **질문**: 가슴압박 시 신체 자세로 옳은 것은? (올바른 자세)
- **정답**: **c) 팔을 곧게 펴고 어깨가 환자 흉부 위에 수직이 되도록 한다.**
- **해설**: 구조자의 어깨가 환자 가슴 바로 위에 오도록 하고, 팔꿈치를 곧게 펴 체중을 실어 압박합니다.

#### Q13. 제세동기 준비 순서

- **질문**: 가슴압박을 교대한 후... 어떤 순서로 제세동기를 준비하고 사용해야 할까요?
- **정답**: **a) A(전원)-C(전극부착)-B(리듬확인)**
- **해설**: 제세동기 사용은 전원을 켜고, 패드를 부착한 뒤, 심전도 리듬을 분석하는 순서로 진행됩니다.

#### Q14. 전극 부착 위치

- **질문**: 전극(electrode)를 성인 환자에게 부착할 때 올바른 위치는?
- **정답**: **c) 오른쪽 쇄골 아래, 왼쪽 쇄골 아래와 왼쪽 갈비뼈 아래**
- **해설**: 일반적인 3-lead 모니터링 부착 위치는 우측 쇄골 아래(RA), 좌측 쇄골 아래(LA), 좌측 갈비뼈 아래(LL) 입니다.

#### Q15. 심전도 리듬 확인 (Asystole)

- **질문**: 제세동기 전극 부착 후... 다음 심전도는 무슨 리듬인가요? (Asystole 영상)
- **정답**: **c) A-systole**
- **해설**: 전기적 활동이 없는 상태인 무수축(Asystole) 입니다.

#### Q16. Asystole 대처

- **질문**: Asystole을 확인한 경우에 어떻게 해야 하나요?
- **정답**: **b) 가슴압박을 지속한다.**
- **해설**: 무수축(Asystole)은 충격 필요 리듬이 아니므로, 즉시 고품질 CPR(가슴압박)을 이어서 수행해야 합니다.

#### Q17. Asystole 약물 투여

- **질문**: 심정지 환자(Asystole)에서... 에피네프린은 어떻게 투여해야 할까요?
- **정답**: **b) 가능한 한 빨리 투여하며 3~5분 간격으로 반복 주사한다.**
- **해설**: 제세동 불가 리듬(Asystole, PEA)에서는 에피네프린을 가능한 한 빨리 투여하고, 3~5분마다 반복합니다.

#### Q18. 심전도 리듬 확인 (VF)

- **질문**: 가슴압박 중... 다음 심전도는 무슨 리듬인가요? (VF 영상)
- **정답**: **a) VF (심실세동)**
- **해설**: 심실세동(Ventricular Fibrillation)은 심장이 무질서하게 떨리는 상태로, 즉각적인 제세동이 필요합니다.

#### Q19. VF 대처

- **질문**: V-fib을 확인한 당신은 어떻게 해야 하나요?
- **정답**: **a) 제세동을 준비한다.**
- **해설**: 심실세동(VF)은 Shockable rhythm이므로 즉시 제세동(Defibrillation)을 준비하고 시행해야 합니다.

#### Q20. 제세동 순서

- **질문**: 제세동이 필요할 때 올바른 순서로 묶인 것은?
- **정답**: **a) A-B-C-D**
- **해설**: 패들 준비(젤) -> 충전 -> 안전 확인(물러나세요) -> 쇼크 실행 및 즉시 가슴압박 재개 순서입니다.

#### Q21. VF 에피네프린 투여 시점

- **질문**: 심정지 환자에서 심실세동(VF)이 확인되었습니다. 이때 에피네프린 투여 시점은?
- **정답**: **c) 제세동을 한 뒤 이어지는 가슴압박 시간에 투여하며 3~5분 간격으로 반복 주사한다.**
- **해설**: VF/pVT의 경우 제세동과 CPR이 우선이며, 에피네프린은 2번째 제세동 후에도 순환이 돌아오지 않을 때 투여합니다.

#### Q22. ROSC 징후 확인

- **질문**: 환자의 ROSC를 확인해야 합니다. 다음 중 자발순환 회복(ROSC)의 징후로 가장 올바른 것은?
- **정답**: **b) 자발 맥박이 돌아오고 혈압이 측정되는 것**
- **해설**: 자발순환 회복(ROSC)은 경동맥 등에서 자발적인 맥박이 촉지되고 혈압이 측정되는 상태를 의미합니다.

## 기술 스택

- **Frontend**: Next.js 15, React, TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Video**: Local video files 및 동적 생성 지원

## 실행 방법 (Getting Started)

개발 서버 실행:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
